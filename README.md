# 🎬 Movie App (Django)

## 📌 Description
A Front End-based movie application that displays movie information and allows users to explore movies using a clean UI.

## 🚀 Features
- Movie listing
- Movie details page
- Admin panel
- API integration

## 🛠 Tech Stack
- HTML, CSS
- JS
